Xcode Tips

发表于 2017-09-22 | 分类于 工具集

注释

MARK 、TODO 、BUG

添加警告

常用快捷键

shift + command + o

快速搜索文件，以及搜索方法名定位方法实现代码

control + 6

快速定位想编辑的方法

列出当前文件中所有的方法，可以通过输入关键字来过滤

shift + command + j

打开对应文件的目录

control + i

方法名中：上下对齐排版

option + command + 左右键

收起 & 展开 代码段

control + command + 上下键

在.h和.m文件之间切换

command + return

切换成 standsrd editor

command + option + return

切换成 assistan editor

左侧、右侧、底部工具栏

command + shift + Y

切换底部 Console VIew 显示或者隐藏

command + 0

切换左边导航区（Navigator）显示或者隐藏

command + option + 0

切换右边工具区（Utility）显示或者隐藏

Tab栏

command + T

新建一个Tab栏

command + shift + [ 或者 ]

Tab栏左右切换

开启无线调试

前提： iOS 11 + Xcode 9 以上设备

步骤：

  • 用数据线链接设备
  • Window -> Device and Simulators
  • 找到对应设备，勾选 Connect via network
  • 断开数据线，运行即可。

只需操作一次，之后开启Xcode会自动连接。

持续整理更新。。。
