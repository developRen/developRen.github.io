快捷键

快速搜索项目中的文件

shift + command + o

打开对应文件的目录

shift + command + j

方法名中：上下对齐排版

control + i

收起 展开 代码段

option + command + 左右键

开启无线调试

前提： iOS 11 + Xcode 9 以上设备

步骤：

  • 用数据线链接设备
  • Window -> Device and Simulators
  • 找到对应设备，勾选 Connect via network
  • 断开数据线，运行即可。

只需操作一次，之后开启Xcode会自动连接。
